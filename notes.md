App:
    Header:
        * title
        * nav bar to other pages (preview and home page)
    Main:
        fetch api endpoints using useEffect()
        display videos:
            *render list of videos that have a link to preview page 
        thumbnail preview:
            *render preview of selected video 
                *use logic from java centroid 
        threshold slider and color picker:
            *render target color and threshold slider
                use logic from java centroid
        submit:
            *render submit button
            *catch CSV with static URL returned by API
        loading:
            *display progress of CSV file and render link to CSV
            *use returned jobID to track status of CSV
